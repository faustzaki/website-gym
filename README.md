# 🏋️‍♂️ GymPro - Modern Fitness Management System

[![React](https://img.shields.io/badge/Frontend-React-61DAFB?style=for-the-badge&logo=react)](https://reactjs.org/)
[![Vite](https://img.shields.io/badge/Bundler-Vite-646CFF?style=for-the-badge&logo=vite)](https://vitejs.dev/)
[![Tailwind CSS](https://img.shields.io/badge/Styling-Tailwind-38B2AC?style=for-the-badge&logo=tailwind-css)](https://tailwindcss.com/)
[![Laravel](https://img.shields.io/badge/Backend-Laravel-FF2D20?style=for-the-badge&logo=laravel)](https://laravel.com/)

**GymPro** adalah platform manajemen gym modern yang dirancang untuk memberikan pengalaman pengguna yang luar biasa. Website ini fokus pada kemudahan akses informasi jadwal latihan, kalkulasi kebugaran (BMI), dan pendaftaran member.

---

## ✨ Fitur Utama

- 🚀 **Premium Hero Section**: Tampilan beranda interaktif dengan efek parallax dinamis.
- 📊 **Interactive BMI Calculator**: Alat hitung massa tubuh otomatis dengan feedback kesehatan instan.
- 👟 **Classes & Trainers Showcase**: Daftar kelas dan pelatih dengan UI modern menggunakan Radix UI (shadcn/ui).
- 📱 **Fully Responsive**: Dioptimalkan untuk pengalaman mulus di mobile, tablet, dan desktop.
- 🎨 **Dark/Light Mode**: Mendukung tema visual yang menyesuaikan preferensi pengguna atau sistem.

---

## 🛠️ Tech Stack

### Frontend
- **Framework**: React.js (Vite)
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui (Radix UI + Lucide Icons)
- **State Management**: Zustand
- **Animations**: CSS Transitions & Micro-interactions

### Backend (Coming Soon)
- **Framework**: Laravel 11
- **API**: REST API with Sanctum Authentication
- **Database**: MySQL

---

## 🚀 Memulai Proyek

### Prasyarat
- Node.js (v18+)
- NPM atau Yarn

### Instalasi Frontend
1. Masuk ke folder frontend:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Jalankan development server:
   ```bash
   npm run dev
   ```
